# REST API Guide

OpenQuery includes a FastAPI-based REST API for programmatic access.

## Starting the Server

```bash
# Install server dependencies
pip install "openquery[serve]"

# Start with defaults (0.0.0.0:8000)
openquery serve

# Custom host and port
openquery serve --host 127.0.0.1 --port 3000
```

### With Docker

```bash
docker compose up
```

This starts the API server with Redis caching on port 8000.

## Endpoints

### POST /api/v1/query

Query a data source.

**Request:**

```json
{
  "source": "co.simit",
  "document_type": "cedula",
  "document_number": "12345678",
  "audit": false
}
```

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `source` | string | yes | Source identifier (e.g., `co.simit`) |
| `document_type` | string | yes | `cedula`, `placa`, or `vin` |
| `document_number` | string | yes | The document value to query |
| `audit` | boolean | no | Enable audit evidence capture |

**Response (200):**

```json
{
  "ok": true,
  "source": "co.simit",
  "queried_at": "2026-03-31T10:30:00Z",
  "cached": false,
  "latency_ms": 4523,
  "data": {
    "comparendos": 0,
    "multas": 0,
    "total_deuda": 0.0,
    "paz_y_salvo": true
  },
  "audit": null
}
```

When `audit: true`, the `audit` field contains the full audit record with screenshots (base64), network log, and PDF report (base64).

**Response (error):**

```json
{
  "ok": false,
  "source": "co.runt",
  "error": "CaptchaError: All solvers failed",
  "latency_ms": 15234,
  "data": null
}
```

## Authentication

Set `OPENQUERY_API_KEY` to require authentication:

```bash
export OPENQUERY_API_KEY="my-secret-key"
openquery serve
```

Then include the key in requests:

```bash
curl -X POST http://localhost:8000/api/v1/query \
  -H "Authorization: Bearer my-secret-key" \
  -H "Content-Type: application/json" \
  -d '{"source": "co.simit", "document_type": "cedula", "document_number": "12345678"}'
```

## Caching

Configure caching to avoid redundant queries:

```bash
# In-memory (default)
export OPENQUERY_CACHE_BACKEND=memory
export OPENQUERY_CACHE_TTL_DEFAULT=3600

# Redis
export OPENQUERY_CACHE_BACKEND=redis
export OPENQUERY_REDIS_URL=redis://localhost:6379/0
```

Cached responses include `"cached": true` in the response.

## Rate Limiting

Each source has a configured rate limit (requests per minute). The API respects these limits automatically. If you exceed the limit, the request will be delayed (not rejected).

| Source | Rate Limit |
|--------|-----------|
| co.simit | 10 rpm |
| co.runt | 5 rpm |
| co.procuraduria | 5 rpm |
| co.policia | 10 rpm |
| co.adres | 10 rpm |
| co.pico_y_placa | 10 rpm |
| co.peajes | 30 rpm |
| co.combustible | 30 rpm |
| co.estaciones_ev | 30 rpm |
| co.siniestralidad | 30 rpm |
| co.vehiculos | 30 rpm |
| co.fasecolda | 10 rpm |
| co.recalls | 5 rpm |

## Example: Python Client

```python
import httpx

client = httpx.Client(
    base_url="http://localhost:8000/api/v1",
    headers={"Authorization": "Bearer my-secret-key"},
    timeout=60.0,
)

# Query SIMIT
resp = client.post("/query", json={
    "source": "co.simit",
    "document_type": "cedula",
    "document_number": "12345678",
})
data = resp.json()
if data["ok"]:
    print(f"Fines: {data['data']['multas']}")
    print(f"Debt: {data['data']['total_deuda']}")
```

## Example: Batch Queries

```python
import asyncio
import httpx

async def query_all(cedula: str):
    async with httpx.AsyncClient(
        base_url="http://localhost:8000/api/v1",
        timeout=60.0,
    ) as client:
        sources = ["co.simit", "co.procuraduria", "co.policia", "co.adres"]
        tasks = [
            client.post("/query", json={
                "source": source,
                "document_type": "cedula",
                "document_number": cedula,
            })
            for source in sources
        ]
        responses = await asyncio.gather(*tasks)
        return {
            source: resp.json()
            for source, resp in zip(sources, responses)
        }

results = asyncio.run(query_all("12345678"))
for source, data in results.items():
    print(f"{source}: {'OK' if data['ok'] else 'ERROR'}")
```

## Deployment

### Docker Compose (recommended)

```yaml
# docker-compose.yml
services:
  openquery:
    build: .
    ports:
      - "8000:8000"
    environment:
      - OPENQUERY_API_KEY=my-secret-key
      - OPENQUERY_CACHE_BACKEND=redis
      - OPENQUERY_REDIS_URL=redis://redis:6379/0
    depends_on:
      - redis

  redis:
    image: redis:7-alpine
    volumes:
      - redis_data:/data

volumes:
  redis_data:
```

### Behind a Reverse Proxy

```nginx
server {
    listen 443 ssl;
    server_name api.example.com;

    location / {
        proxy_pass http://127.0.0.1:8000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_read_timeout 120s;  # Queries can take 30-60s
    }
}
```
